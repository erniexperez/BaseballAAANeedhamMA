# Pitch Tracker

A web app for tracking youth baseball pitch counts, enforcing rest rules, and supporting multi-user cloud access.

## Features
- Record pitch counts for all games
- Enforce youth baseball rest rules
- Responsive, mobile-friendly UI
- Cloud backend (PostgreSQL/Render-ready)
- Multi-user support

## Deployment
1. Clone this repo to your Render-connected GitHub account.
2. Set the `DATABASE_URL` environment variable in Render (PostgreSQL connection string).
3. Deploy the web service on Render.
4. Access from any device/browser.

## Local Development
- `npm install`
- `npm start`
